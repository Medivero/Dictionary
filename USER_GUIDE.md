# Dictionary

Пользовательская документация приложения словаря.

## Что это за приложение

`Dictionary` помогает хранить список английских слов и их переводов на русский язык.

В приложении можно:

- просматривать сохраненные слова;
- добавлять новые слова;
- автоматически получать перевод при вводе нового слова;
- редактировать существующие записи;
- удалять слова;
- искать слова по названию или переводу.

## Как запустить

### Что нужно

- установленный `Node.js`;
- установленный `npm`.

### Первый запуск

В каталоге проекта выполните:

```bash
npm install
npm run dev
```

После запуска откройте в браузере:

```text
http://localhost:5173
```

## Как пользоваться

### Просмотр словаря

После открытия приложения словарь загружается автоматически. В верхней части экрана отображается количество сохраненных слов.

### Добавление нового слова

В первой строке таблицы:

1. Введите английское слово в поле `Слово`.
2. Подождите около 1 секунды.
3. Приложение автоматически подставит перевод в поле `Перевод`.
4. Нажмите кнопку сохранения.

После сохранения слово появится в списке.

### Редактирование слова

Чтобы изменить запись:

1. Найдите нужное слово в таблице.
2. Измените текст в строке.
3. Нажмите кнопку сохранения.

После этого приложение покажет уведомление об успешном изменении.

### Удаление слова

Чтобы удалить запись:

1. Найдите нужную строку.
2. Нажмите кнопку удаления рядом со словом.

Слово будет удалено из словаря сразу после нажатия.

### Поиск

В нижней части экрана находится поле `Поиск`.

Поиск:

- выполняется автоматически при вводе текста;
- ищет совпадения и по слову, и по переводу;
- не зависит от регистра букв.

## Уведомления

После сохранения или редактирования приложение показывает короткое всплывающее сообщение.

## Если словарь пуст

Если в словаре нет записей, на экране появится сообщение о пустом словаре и предложение добавить новое слово.

## Ограничения

- Приложение рассчитано на узкие экраны. На экранах шире `500px` основной интерфейс не отображается.
- Перевод нового слова подставляется автоматически через внешний сервис.
- Поле перевода в строке добавления недоступно для ручного ввода.
- Для корректной работы автоперевода и сохранения слов сервер приложения должен быть запущен.

## Частые проблемы

### Приложение не открывается

Проверьте, что команда `npm run dev` выполнена без ошибок и страница открыта по адресу `http://localhost:5173`.

### Слова не сохраняются

Проверьте, что сервер запущен вместе с клиентом. В этом проекте он стартует той же командой `npm run dev`.

### Перевод не подставляется

Скорее всего недоступен внешний сервис перевода или сервер приложения не отвечает.
